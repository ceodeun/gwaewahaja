# 과(외하)자 — 과외 연결 플랫폼

나에게 딱 맞는 과외 선생님을 찾아주는 서비스

## 시작하기

```bash
npm install
npm run dev
```

브라우저에서 http://localhost:3000 열기

## 배포 (Vercel)

1. GitHub에 이 폴더 올리기
2. vercel.com 가서 Import
3. 자동 배포 완료!

## 주요 기능

- 랜딩 페이지 (역할 선택)
- 회원가입 / 로그인
- 대시보드 (홈 · 선생님 찾기 · 수업 일정 · 채팅)
- 선생님 목록 및 예약

## 기술 스택

- Next.js 14 (App Router)
- TypeScript
- CSS-in-JS (inline styles)
